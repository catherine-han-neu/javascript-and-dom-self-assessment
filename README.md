### Javascript and DOM Self Assessment

This page uses AJAX (fetch/await) to fetch data from airbnb_sf_listings_500.json as provided by the professor, then loads and displays the first 50 listings from the JSON file. 

It shows listing name, description, amenities, host (name and photo), price, thumbnail, and some creative addition that separates your code from others (in my case, a card-like appearance view using box-shadows). 

This project is deployed using github-pages at: https://catherine-han-neu.github.io/javascript-and-dom-self-assessment/
